# analyze-reverse-shell-detector
Detects reverse shell connections by analyzing network traffic patterns and established connections, flagging suspicious shell-like activity based on communication frequency and data transfer characteristics. - Focused on Data analysis and reporting

## Install
`git clone https://github.com/ShadowGuardAI/analyze-reverse-shell-detector`

## Usage
`./analyze-reverse-shell-detector [params]`

## Parameters
- `-h`: Show help message and exit
- `-i`: No description provided
- `-o`: Path to save the analysis report. Defaults to reverse_shell_report.txt
- `-t`: No description provided
- `-f`: Format of the input file. Currently supports
